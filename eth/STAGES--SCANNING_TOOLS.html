SCANNING TOOLS

Pre-requisites:
TCP (transmission control protocol)
• Connection-oriented
• Handshake ✔
• Used on applications requiring high reliability
• Ex: HTTP,FTP,Telnet
[SYN],[SYN,ACK],[ACK]..CLIENT HELLO



UDP (user datagram protocol)
• Connectionless
• No handshake
• Used on applications requiring fast connection
• Ex: DNS,DHCP,SNMP
