VPN

Virtual Private Network : A technology that allows for the extension of a private or local network to hosts that might not be on that local network.
VPN tunneling Protocol: Encrypted data packet with five layer data packets
Two-factor authentication : became common from vpn. A technique where more than just a username and password are reqd to authenticate.

PROXIES : intermediatry btwn client and main server
A server that acts on behalf of a client in order to access another service.
• Anonymity
• Security
• Content Filtering
• Increased performance

Web Proxies : built for web traffic .
