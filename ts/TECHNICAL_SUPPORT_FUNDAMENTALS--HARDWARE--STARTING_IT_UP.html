STARTING IT UP

BIOS - basic input output services
BIOS is installed on the Read-only Memory Chip in the motherboard.
UEFI : Unified Extensible Firmware Interface . Checks for external hardware .
beep sound while starting the computer. It's the Power On Self Test (POST) -figures out what hardware is connected. then bios comes into play.
Single beep - successful
Double beep - POST error
