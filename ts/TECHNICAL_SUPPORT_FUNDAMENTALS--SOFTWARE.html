SOFTWARE

Software : It is the intangible instructions that tell the hardware what to do!

Coding : Transalating from one language to another. building application

Scripting : Coding in a scripting language . Scripts perform single range tasks.

Programming : Coding in programming language. (music player, text editor)

Types :
• Application : A software created to fulfil a specific need - text editor, web browser
• System : A software used to keep our core system running, like OS tools and utilities
• Firmware : Permanently stored on computer . (EX.BIOS , UEFI)

Software versions : follow sequential numbering
